# nuit-de-l-info-Hair_gonomie
Village Cartoon & Bibliothèque Interactive
Description
Deux applications web interactives basées sur l'univers d'Astérix & Obélix, permettant de naviguer dans des environnements éducatifs de manière ludique.

Fonctionnalités
1. Village Cartoon Fullscreen
Interface : Village gaulois en plein écran avec 4 huttes principales

Huttes interactives :

Cours (liée à cours.html)

Devoirs (en construction)

Messages (en construction)

Ressources (en construction)

Personnages animés :

Astérix avec animation de flottement

Obélix avec animation de rebond

Getafix en mouvement circulaire rapide

Système de cartes vintage :

Roadmap de formation (Astérix)

Guide du village (Obélix)

Effets visuels avancés (papier vieilli, sceaux de cire)

Chien interactif : Avec barre d'énergie au survol

2. Bibliothèque Interactive
Environnement : Bibliothèque magique avec fond réaliste

Livres éducatifs :

Mathématiques

Physique

Histoire

Programmation

Potions magiques (en haut à gauche) :

Potion de Sagesse

Potion de Mémoire

Potion de Concentration

Personnages :

Bibliothécaire (Getafix)

Étudiant (Astérix)

Système d'information : Cartes détaillées pour chaque élément

Installation et utilisation
Prérequis
Navigateur web moderne (Chrome, Firefox, Edge, Safari)

Serveur web local (optionnel)

Installation simple
Téléchargez tous les fichiers

Placez-les dans un même dossier

Assurez-vous que les images sont dans le même dossier que les fichiers HTML

Ouvrez index.html dans votre navigateur
